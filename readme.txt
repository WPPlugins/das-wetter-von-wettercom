Das Wetter Plugin für Wordpress von wetter.com zeigt aktuelle Wetterinformationen für die Stadt deiner Wahl an. Das Plugin ist leicht zu installieren und bietet verschiedene Anpassungsmöglichkeiten an.

== Description ==

Mehr Informationen zum Plugin finden Sie unter http://www.wetter.com/services/wetter_tools/wordpress_plugin/. Damit Sie die Wetterdaten abrufen können, benötigen Sie einen Zugang zu unserer openweather Schnittstelle.

The current version of this plugin is only available in German.

== Installation ==

Eine ausführliche Installationsanleitung ist unter http://www.wetter.com/services/wetter_tools/wordpress_plugin/ zu finden.
